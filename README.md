# stickypostrr
a stickypost app made with ruby

## Installation
### Gem
```sh
$ gem install StickyPostRR
```

### Source
```sh
$ git clone https://github.com/NillyTheL0L/stickypostrr
```
```sh
$ rake build
```

## Usage
### Executable
```sh
$ stickypostrr
```

### File
```sh
require 'stickypostrr'
Post.new
```
